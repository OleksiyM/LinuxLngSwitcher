Quick Setup

1

Install Dependencies

Make sure GTK4 and Libadwaita development libraries are installed:

# Ubuntu / Debian
sudo apt install libgtk-4-dev libadwaita-1-dev

# Fedora
sudo dnf install gtk4-devel libadwaita-devel
2

Download and Grant Permissions

Download the binary and add your user to the input group to read keyboard events:

wget https://github.com/OleksiyM/LinuxLngSwitcher/releases/download/latest/gnome-lng-switcher
chmod +x gnome-lng-switcher
sudo usermod -aG input $USER

⚠️ You must Log Out and log back in to apply group permissions.

3

Run and Configure

Launch the settings window, click «Enable Helper» to install the status bar menu, and start the daemon:

./gnome-lng-switcher
